== Description ==

Voegt de Cartel Auto-Webshop toe aan de website die alle voorraad auto's laat zien.

**Shortcode**

Gebruik onderstaande shortcode om de module te laten tonen.
`
[autowebshop configuratiecode="1"]
`
De configuratie code kan opgevraagd worden bij [Cartel](mailto:support@cartel.nl)

**Variablen**

Onderstaande variabelen kunnen gebruikt worden in de shortcode.

* `configuratiecode` Voorbeeld: `configuratiecode="1"` deze code is op te vragen bij [Cartel](mailto:support@cartel.nl)
* `voertuigsoort` Voorbeeld: Dient komma gescheiden toegevoegd te worden `voertuigsoort="occasion,nieuw"`
de volgende waarde zijn beschikbaar `occasion` `nieuw` `demo`
* `carrosserie` Voorbeeld: Dient komma gescheiden toegevoegd te worden `carrosserie="hatchback,terreinwagen"`
* `voertuigcategorie` Voorbeeld: `voertuigcategorie="personenauto"` de volgende waarde zijn beschikbaar
`personenauto` `bedrijfswagen`
* `voertuigmax` Voorbeeld: `voertuigmax="48"` de volgende waarde zijn beschikbaar `12` `24` `48` `96`
* `sortering` Voorbeeld: `sortering="bouwjaar-aflopend"` de volgende waarde zijn beschikbaar
`prijs-oplopend` `prijs-aflopend` `ellerstand-oplopend` `bouwjaar-aflopend` `laatst-toegevoegd`
* `bouwjaar` Voorbeeld: `bouwjaar="2000-2008"`
* `tellerstand` Voorbeeld: `tellerstand="1000-68000"`
* `merkmodelselectie` Voorbeeld: `merkmodelselectie="volkswagen|golf"` of alleen merk `merkmodelselectie="volkswagen"`
* `prijs` Voorbeeld: `prijs="1000-25000"`
